O que são Headers?

A Tradução para Header é "Cabeçalho". Normalmente utilizado para dar destaque em uma sessão ou iniciar um conteúdo. Por exemplo, nesta página, no início da foi usado um header h1 para destacar o texto "Desafio HTML" e um header h4 para o ínidice. Headers vão de h1 a h6 e conforme o número aumenta 1,2,3,... o tamanho do cabeçalho diminui, ou seja, o texto dentro de tags h1 são maiores do que nos textos dentro de tags h2 e assim por diante.
No arquivo html o header ocupa todo a largura daquela linha em que ele está, na linguagem técnica ele cria um block-level, ou seja, um bloco que ocupa todo aquele espaço onde ele está inserido. A imagem abaixo ilustra o conceito de block-level.

Quais são as principais tags de texto?

o html oferece uma gama de tags que podem ser usadas nos textos. Tags como < del > que grifam sobre a palavra ou frase, ou até mesmo quando você quer usar uma abreviação e explicá-la logo em seguida, por exemplo EUA (Estados Unidos da América)

e toda vez que eu citar EUA colocando < abbr > ela irá destacar. Claro que no caso da abreviação o uso CSS facilitará bastante.

  • segundo item
‹li› elemento da lista utlizado em conjunto com ‹ul› ou ‹ol› ‹a› utilizado para hyperlinks
‹hr› representa uma linha de corte
‹sub› utilizado por exemplo para uma fórmula de química C8H10N4O2, ‹sup› o oposto do ‹sub›, os elementos ficarão em cima 122.
